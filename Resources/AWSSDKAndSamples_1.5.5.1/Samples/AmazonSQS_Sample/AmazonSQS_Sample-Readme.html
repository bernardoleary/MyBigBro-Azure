Amazon SQS Sample

This is a sample that demonstrates how to make basic requests to Amazon SQS using the AWS SDK for .NET.

Prerequisites

  • You must have a valid Amazon Web Services developer account.
  • Requires the AWS SDK for .NET. For more information on the AWS SDK for .NET, see http://aws.amazon.com/sdkfornet.
  • You must be signed up to use Amazon SQS. For more information on Amazon SQS, see http://aws.amazon.com/sqs.

Running the Sample

The basic steps for running the Amazon SQS sample are:

  1. Open the AmazonSQS_Sample.sln file in Visual Studio.
  2. Open the App.config file.
  3. Enter your Access Key ID and Secret Access Key:
    <add key="AWSAccessKey" value="<Your Access Key ID>"/>   
<add key="AWSSecretKey" value="<Your Secret Access Key>"/>
  4. Save the file.
  5. Run the sample in Debug mode by typing F5.

See the Amazon SQS Getting Started Guide for step-by-step instructions on running this sample.
